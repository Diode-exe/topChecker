# topChecker

Check all 4 letter domains with the TLD of .top

If the domain is available, it will be added to the reachable.txt file. If the domain is not available, it will skipped.

## Usage

1. Clone the repository

2. Install the required dependencies using pip install -r requirements.txt

3. Run the script using ```python topChecker.py```

4. The available domains will be saved in the ```reachable.txt``` file.

## Note

- This script takes some time (days) to run as it checks all 4 letter combinations with the .top TLD. Please be patient while it is running.
